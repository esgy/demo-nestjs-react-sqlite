# Demo App w/ NestJS, Sqlite, React

Simple Demo app with local state.

## Setup

`npm i`

Duplicate or rename .env.example to .env.local in the `frontend` package

## Test

For Unit tests:

`npm test`

For E2E tests:

`npm run cypress`
